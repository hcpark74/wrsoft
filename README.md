# 🚀 WRSoft Website Upgrade Project

우리소프트(WRSoft)의 23년 기술력과 신뢰를 시각적으로 증명하고, 기술 리더십을 강조하기 위한 웹사이트 고도화 프로젝트입니다.

## 🎯 핵심 목표
- **신뢰의 시각화**: '23년 무사고', '개발자 비율 90%' 등의 수치를 인포그래픽으로 증명
- **기술 리더십**: AI 및 금융/공공 프로젝트의 아키텍처와 사례 중심의 구체적 노출
- **개발 문화 브랜딩**: 개발자가 선호하는 기술 스택과 문화를 강조하여 우수 인재 영입

---

## 📋 페이지별 업그레이드 전략

### 1. 메인 페이지 (`index.html`)
- **수치 시각화**: 텍스트 위주의 지표를 Animated Counter와 인포그래픽 아이콘으로 교체
- **고객사 신뢰**: 로고 그리드를 개선하여 금융/공공 섹터의 압도적 포트폴리오 강조
- **비주얼 현대화**: 추상적 이미지 대신 실제 업무 현장 및 기술 모션 그래픽 적용

### 2. 주요사업 & 솔루션 (`sub/business.html`, `sub/solution.html`)
- **기술 스택 구체화**: 영역별 사용 기술(Java, Spring Boot, MSA 등) 아이콘 배치
- **AI Case Study**: 기업평가 고도화 등 실제 AI 적용 사례를 아키텍처 중심으로 설명
- **프로세스 시각화**: 솔루션 도입 단계를 Step-by-Step 카드 UI로 직관적 구성

### 3. 수행사업 (`sub/reference.html`)
- **데이터 관리 체계화**: 1,500줄의 중복 코드를 제거하고 JSON 데이터 기반 동적 렌더링 도입
- **스마트 검색/필터**: 키워드 기반 검색 기능 및 도메인별(금융/공공/기타) 상세 필터 구현
- **성과 중심 스토리텔링**: [Problem-Solution-Benefit] 구조의 프로젝트 상세 정보 제공

### 4. 인재채용 (`sub/recruit.html`)
- **개발자 중심 문화**: 기술 세미나, 코드 리뷰 등 실제 성장 환경을 시각적 콘텐츠로 보강
- **기술 스택 투명성**: GitLab, Jira, Slack 등 실제 협업 툴과 개발 환경 정보를 노출
- **채용 프로세스 고도화**: 직무별 역량을 선택하고 문의할 수 있는 전문적인 폼 인터페이스

---

## 🛠 기술적 최적화 로드맵

1. **공통 컴포넌트 라이브러리 (Lego)**
   - Header, Footer 등 중복 코드 분리 및 동적 로드 (PHP 또는 JS Fetch 활용)
2. **코드 슬리밍 (Slimming)**
   - `reference.html` 등 거대 페이지의 데이터를 외부 JSON화하여 유지보수성 확보
3. **성능 및 반응형 최적화**
   - 인라인 SVG 자산화 및 미디어 쿼리 기반의 통합 반응형 코드 구조로 개편 (Current: `only-pc` 분기 구조 탈피)
4. **SEO 및 접근성 강화**
   - 시맨틱 마크업 강화 및 메타 데이터 최적화로 브랜드 노출도 상향

🚀 기술적 최적화 제안 (Systemic Upgrade)
공통 컴포넌트화: PHP나 간단한 Template 엔진, 또는 JS fetch를 이용해 헤더/푸터를 분리 관리하여 수정 시 모든 페이지에 한 번에 반영되도록 합니다.
반응형(Responsive) 디테일 개선: 현재 소스에 only-pc, no-pc-section으로 분리된 구간이 많습니다. CSS Media Queries를 적극 활용해 하나의 코드로 유지되도록 최적화가 필요합니다.
성능 최적화: 현재 인라인 SVG가 매우 방대하여 로딩 속도에 영향을 줄 수 있습니다. 외부 에셋화하거나 Symbol 방식으로 관리하여 페이지 로딩 성능을 높이십시오.

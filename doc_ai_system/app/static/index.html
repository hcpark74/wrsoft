대화

대화 범위를 조정하면서 답변 품질을 확인합니다.

준비됨
현재 선택한 카테고리 범위 안에서 문서를 검색해 답변합니다.

현재 범위 안내

대화는 선택한 카테고리 범위 안에서만 문서를 검색합니다.

  • 필터를 비우면 전체 문서를 대상으로 답변합니다.
  • 카테고리를 고르면 해당 분류 문서만 검색합니다.
  • 참조 문서를 보고 응답 품질을 점검할 수 있습니다.

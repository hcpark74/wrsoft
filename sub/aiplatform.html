KEY FEATURES

주요 기능 상세

01

하이브리드 검색 + 재순위화

pgvector 벡터 검색과 tsvector 키워드 검색을 RRF 알고리즘으로 결합하고 Cross-Encoder로 최종 순위를 조정합니다.

02

SSE 실시간 스트리밍

Server-Sent Events 기반 토큰 스트리밍으로 답변이 생성되는 즉시 화면에 표시 — 체감 속도 50% 이상 개선.

03

Text-to-SQL 분석

"지난달 A부서 매출 얼마야?" 자연어 질문을 SQL로 자동 변환하여 DB에서 정확한 수치를 추출합니다.

04

출처 인용 정보

응답과 함께 참조된 문서의 출처 정보를 제공하여 답변 신뢰도를 높입니다.

05

한국어 OCR (PaddleOCR)

PaddleOCR 기반 한국어 OCR 93~95% 정확도, GPU 가속으로 PDF·이미지 내 텍스트를 자동 추출합니다.

06

AI 운영 에이전트

"지금 서버 상태 어때?" — AI 에이전트가 로그를 분석하고 장애 조치까지 제안하는 대화형 운영 환경입니다.
